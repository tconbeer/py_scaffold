# How to use

1. Copy the contents of /src into a new directory (`~/my_project`)
1. `cd ~/my_project`
1. `git init .`
1. `pipenv install`
1. `pipenv run pre-commit install`
1. `git add .`
1. `git commit -m "first commit"`
1. `git branch -M main`
1. `gh repo create`
1. `git push --set-upstream origin main`
